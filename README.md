# 📊 Sales Analytics Dashboard (Power BI)

## 🚀 Project Overview  
This project showcases an interactive **Sales Analytics Dashboard** built using **Power BI**.  
It provides valuable insights into sales performance, customer behavior, product trends, and geographical distribution.

The goal is to help businesses make **data-driven decisions** using clear and interactive visualizations.

---

## 📌 Key Features  

### 📈 Executive Sales Overview
- Total Sales: ₹10.03M  
- Total Quantity: 99K  
- Total Orders: 307  
- Average Order Value: ₹32.68K  
- Sales trend analysis over time  

### 📦 Product Performance Analysis
- Sales by product line (Classic Cars, Vintage Cars, Motorcycles, etc.)  
- Quantity distribution by deal size  
- Product-wise revenue comparison  

### 🌍 Geographical Sales Insights
- Sales distribution across countries  
- Territory-wise performance (EMEA, NA, APAC)  
- Map-based visualization  

### 👥 Customer Insights
- Unique Customers: 92  
- Revenue per customer  
- Top customers by sales and orders  

### 💰 Deal Size Analysis
- Sales contribution by deal size (Small, Medium, Large)  
- Average order value comparison
